Frontend Roadmap

Frontend Development is the development of the graphical user interface of the website, through the use of HTML , CSS , Javascript, so that the user can view and interact with the website.

  • HTML: The Hyper Text Markup Language or HTML is a standard markup language for document designed to be displayed in a web browser.


  • CSS: Cascading Style sheet or (CSS) is a style sheet language used for describing the presentation of a document written in a markup language such as HTML or XML.



    1. Pure CSS

    2. Tailwind (framework)

    3. Bootstrap (framework)

    4. Sass (Preprocessor)



  • JavaScript: Javascript is often abbrevative as JS, is a programming language that is one of the core technologies of world wide web, along side HTML and CSS. It is used to add functionalities in the website.



    1. Vanila JS

    2. jQuery (Library)

    3. React (framework)

    4. Angular (framework)

    5. Vue (framework)

    6. TypeScript (Preprocessor)
